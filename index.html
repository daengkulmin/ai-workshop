📚 Curriculum

Workshop Modules

4 modules ครอบคลุมทุก aspect ของการ Coding with AI

🧠
MODULE 01

Prompt Engineering for Devs

เรียนรู้หลักการเขียน prompt ที่ได้ผลสำหรับงาน coding โดยเฉพาะ

  • System prompts และ context setting
  • Few-shot examples สำหรับ code tasks
  • Chain-of-thought สำหรับ debugging
  • Structured output (JSON, TypeScript types)
⚡
MODULE 02

AI-Assisted Coding

เพิ่ม velocity ด้วย AI autocomplete, generation, และ refactoring

  • GitHub Copilot inline suggestions
  • Claude Code สำหรับ agentic tasks
  • Cursor IDE features
  • Test generation อัตโนมัติ
🔍
MODULE 03

AI Code Review & Debug

ใช้ AI เป็น pair programmer สำหรับ review และ debug

  • AI-assisted PR review
  • Bug hunting ด้วย context window
  • Security vulnerability detection
  • Performance optimization suggestions
🔧
MODULE 04

Building AI-Powered Features

Integrate AI เข้า application จริงด้วย APIs และ patterns ที่ใช้งานได้

  • Anthropic API & OpenAI API basics
  • RAG (Retrieval-Augmented Generation)
  • Function calling และ tool use
  • Streaming responses
example — claude_api.py
Module 04
import anthropic

client = anthropic.Anthropic()

# สร้าง completion ด้วย Claude API
message = client.messages.create(
  model="claude-opus-4-6",
  max_tokens=1024,
  messages=[
    {"role": "user", "content": "Review this code and suggest improvements"}
  ]
)

print(message.content[0].text)

Workshop Labs

6 labs ที่ลงมือทำจริง ตั้งแต่ง่ายไปยาก

01

Setup AI Development Environment

ติดตั้งและ configure Claude Code, GitHub Copilot, และ API keys สำหรับ workshop

Beginner ~20 min
02

Prompt Patterns for Code Generation

ฝึกเขียน prompts แบบต่างๆ เพื่อ generate function, class, และ module ที่ใช้งานได้จริง

Beginner ~30 min
03

Refactor Legacy Code with AI

นำโค้ดเก่าที่ให้มา ใช้ AI ช่วย refactor ให้ clean, readable, และ testable

Intermediate ~30 min
04

AI-Powered Test Suite

Generate unit tests, edge cases, และ integration tests ด้วย AI จาก codebase ที่มีอยู่

Intermediate ~30 min
05

Build a Code Review Bot

สร้าง script ที่รับ git diff แล้วใช้ Claude API เพื่อ review code และ output markdown report

Advanced ~45 min
06

RAG-Powered Documentation Q&A

สร้าง mini-app ที่ embed documentation แล้วให้ถาม-ตอบด้วย natural language

Advanced ~45 min
